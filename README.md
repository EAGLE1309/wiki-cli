## To use it do:
Seamlessly by using npx:
```
npx wikipedia-cli
```

Manual setup:
```
git clone https://github.com/EAGLE1309/wikipedia-cli
```
```
cd wikipedia-cli
```
```
npm i
```
```
node .
```
